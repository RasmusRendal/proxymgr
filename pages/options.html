Domain Rule

This browser extension is rather simple. In the proxies tab, you can define as many proxies as you'd like. By default, all pages will then be loaded by the highest proxy. Then, using the extension toolbar in your browser you can click the proxymgr icon to show the rules for the current website. This allows you to set which proxy should be used for loading each site. Currently, the rules matching is quite dumb, and selects the first applicable rule. This should be improved in the future.

There also is a feature that allows you to temporarily set the proxy for all requests sent in the tab. This is useful if you're using a VPN, but want to view a VPN-blocking site real quick, an issue I've had quite often with other proxy manager addons.
